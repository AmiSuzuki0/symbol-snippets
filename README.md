# symbol-snippet

HTML の記号入力が覚えてなくて探して入力するのが面倒臭いので作りました

## 使い方

-   プロジェクトのルートディレクトリの部分に
    `.vscode`のフォルダを置く

-コマンドパレットを開く
`mac : Command + Shift + p`
` Win : Ctrl + Shift + p`

-   スニペットの挿入を選ぶ
    ![alt text](image.png)

-   html,ejs ファイルの際は表示以下が表示される
    ![alt text](image-1.png)

-   入れたい文字を選択すると入力される
